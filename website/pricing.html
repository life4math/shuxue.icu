FAQ

免费版有什么限制？+
免费版可浏览题库（10题/日）、查看知识图谱、基础仪表盘、3次/月深度分析、Blog完整访问。核心浏览功能无期限限制。升级 Pro 可解锁无限使用和 AI 审核。
为什么只面向教师收费？+
shuxue.icu 定位为教研工具平台，不是学科培训机构。双减政策禁止面向 K-12 学生的学科类收费培训。我们为教师提供备课、命题、学情分析工具，完全合规。
Team 版座席如何增购？+
Team 版默认 5 座席。超出部分按 ¥39/座席/月增购，上限 20 座席。超过 20 座席建议联系商务定制 Enterprise 方案。
API 数据如何保证隐私合规？+
所有 API 输出数据经三层脱敏：1) 学生姓名→匿名ID；2) 具体成绩→等级(A/B/C)；3) 班级信息→聚合统计。原始可识别数据不出库。API 许可协议禁止逆向提取和再授权。
年付折扣如何计算？+
年付 = 月价 × 10（免2个月），相当于 17% 折扣。Pro 年付 ¥490（原 ¥588），Team 年付 ¥1990（原 ¥2388）。年付中途可退款（按已使用月份扣减）。
免费用户的数据会被用于 API 吗？+
用户协议中明确告知：平台使用脱敏聚合数据（非个人可识别数据）构建教研数据资产。用户可选择退出数据贡献（设置中开启 Privacy Mode），退出后其数据不纳入 API 数据集。
